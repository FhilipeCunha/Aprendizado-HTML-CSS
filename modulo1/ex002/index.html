Sinonimo de Informações

Informação, é um conjunto de dados organizados e processados que ganham significados, contextos e utilidades. Ela funciona como a base para a criação do conhecimento. Os dados brutos(isolados) transforam-se em informação quando estruturados para responder a uma necessidade ou transmitir uma imagem.">
